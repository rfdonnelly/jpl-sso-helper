= SSO Helper

Allows interactive CLI applications to authenticate with web applications behind SSO.

https://github.com/user-attachments/assets/04325b43-73ce-440d-9317-97716892c995

== How it Works

. Your interactive CLI application calls: `sso-helper <site-url>`
. SSO helper opens the site URL using Chrome
. The user authenticates with SSO interactively
. Once the user has authenticated, SSO helper will print the authentication cookies to STDOUT in YAML and terminate
. Your interactive CLI obtains the authentication cookies from STDOUT
. Your interactive CLI uses the authentication cookies to make requests to the site

== Dependencies

The only dependency is https://docs.astral.sh/uv/[uv]

== Usage Example

[source,python]
----
import requests
import yaml

site_url = ...

session = requests.Session()

# Authenticate with SSO
result = subprocess.run(["uvx.exe", "--from=git+ssh://git@github.com/rfdonnelly/sso-helper.git", "sso-helper", site_url], capture_output=True)
cookies = yaml.safe_load(result.stdout)
domain = urlparse(site_url).netloc
for cookie in cookies:
    if cookie["domain"] == domain:
       session.cookies.set(cookie["name"], cookie["value"], domain=domain, path=cookie["path"])

# Authenticated request
session.get(f"{site_url}/example")
