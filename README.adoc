= JPL SSO Helper

Allows interactive CLI applications to authenticate with web applications behind JPL SSO.

== How it Works

. Your interactive CLI application calls: `jpl-sso-helper <site-url>`
. JPL SSO helper opens the site URL using Chrome
. You authenticate with JPL SSO interactively
. Once you have authenticated, JPL SSO helper will print the authentication cookies to STDOUT in YAML and terminate
. Your interactive CLI obtains the authentication cookies from STDOUT
. Your interactive CLI uses the authentication cookies to make requests to the site

== Install

[source]
----
uv pip install .
----

This will create an executeable in `./.venv` that your interactive CLI can invoke.

== Test

[source]
----
uv run jpl-sso-helper <site-url>
----

== Usage Example

[source,python]
----
import requests
import yaml

session = requests.Session()

# Authenticate with JPL SSO
result = subprocess.run(["jpl-sso-helper", site_url], capture_output=True)
cookies = yaml.safe_load(result.stdout)
domain = urlparse(site_url).netloc
for cookie in cookies:
    if cookie["domain"] == domain:
       session.cookies.set(cookie["name"], cookie["value"], domain=domain, path=cookie["path"])

# Authenticated request
session.get(f"{site_url}/example")
----
